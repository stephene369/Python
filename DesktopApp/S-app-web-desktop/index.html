This desktop application, built with Python, leverages the Argos Translate package to provide offline translation capabilities. It automatically installs the necessary language packages, including "translate-en_fr-1_9.zip," "en_ja.zip," and "translate-fr_en-1_9.zip." With these resources, the application can handle translations between English, French, and Japanese. The Argos Translate models are optimized for performance and accuracy, capable of translating thousands of words in real-time, making it a reliable tool for various translation needs.

Why did I built this ?

During my bachelor's degree, I frequently needed to share files with peers, which often involved using flash drives. Unfortunately, I lost two of my flash drives during this process. This inconvenience sparked the idea to develop an application that enables seamless file sharing between laptops and from laptops to Android devices.

In addition to file sharing, I wanted to address another challenge: language barriers. To solve this, I integrated a feature in the application that leverages the Argos Translate model for offline translations. This was motivated by the lack of internet connectivity options available for translation services in some situations.
Moreover, the application includes two models for creating CVs, allowing users to generate professional resumes efficiently. By combining file sharing, offline translation, and CV creation, the application aims to provide a comprehensive solution for various practical needs.
